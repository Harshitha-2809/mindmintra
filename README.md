# MindMitra

MindMitra is a full-stack mental wellness app with authentication, chat, posts, profile, mood tracking, admin tools, and nearby help features.

## Run locally

```bash
npm install
npm run dev
```

Frontend URL: `http://127.0.0.1:5173`

Backend health URL: `http://127.0.0.1:5000/api/health`

## Environment

Copy `.env.example` to `.env` and update the values before running the app.
